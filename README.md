# 🚀 Java TPs - Génie Informatique (EST)

Ce dépôt contient mes travaux pratiques réalisés en langage **Java** dans le cadre de ma formation à l'EST. Il illustre l'application des concepts fondamentaux de la programmation orientée objet (POO) et de la manipulation de données.

## 📁 Projets Inclus

### 1. Gestion d'Établissement Scolaire (POO)
Application console permettant de gérer les données des étudiants et de calculer leurs résultats académiques.
*   **Concepts clés :** Encapsulation (classes et attributs privés), Tableaux d'objets.
*   **Algorithmique :** Implémentation d'un tri à bulles pour le classement par mérite.
*   **Logique :** Validation des modules avec gestion des notes éliminatoires.

### 2. Calculateur Géométrique
Un programme simple pour le calcul des propriétés des cercles.
*   **Interaction :** Utilisation de la classe `Scanner` pour la saisie utilisateur.
*   **Précision :** Exploitation de la bibliothèque `Math` pour les calculs géométriques.

## 🛠 Environnement Technique
*   **Langage :** Java SE
*   **Outils :** Terminal (CMD), VS Code, Git.
*   **Documentation :** Génération de rapports techniques via `javadoc`.
